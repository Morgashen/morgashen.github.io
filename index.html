About the Project

The Maze Project is a fascinating journey into the world of algorithms. My project demonstrates various maze generation techniques and their corresponding solving algorithms. Dive in to see how it all works!

Features

  • Random Maze Generation
  • Depth-First Search (DFS) Solver
  • Breadth-First Search (BFS) Solver
  • Visual Path Highlighting
  • Interactive Maze Editing
